Proyectos

Sport Tienda

Proyecto desarrollado como parte de una feria de ciencias, enfocado en la implementación de una arquitectura full stack. Se utilizaron las siguientes tecnologías:

  • Vue.js (frontend)
  • Node.js (backend)
  • JavaScript y TypeScript
El frontend fue desplegado en Vercel, mientras que el backend se alojó en Render, permitiendo una experiencia dinámica e interactiva.

Sistema para administracion de jardin

Sistema desarrollado a pedido de la directiva de un jardín de infantes. Se implementó un sitio web con doble funcionalidad: una interfaz pública para que los padres conozcan el jardín, y una plataforma interna para la gestión administrativa.

Entre las funcionalidades del sistema se incluyen:

  • Registro de asistencias
  • Carga de noticias institucionales
  • Notas personalizadas por alumno
  • Gestión de fechas importantes mediante un calendario integrado visible para los padres
Todo el sistema fue diseñado con una interfaz intuitiva, amigable y visualmente atractiva, priorizando la experiencia del usuario.

Tecnologías utilizadas: PHP, HTML5, CSS3, JavaScript, MySQL y Render (para el despliegue del backend).

Gestor de Proyectos

Sistema orientado a la gestión de equipos dentro de una empresa. Se implementó un sistema de roles que define los permisos y acciones disponibles para cada usuario, promoviendo una estructura jerárquica clara y segura.

El gestor permite crear y organizar proyectos mediante un diagrama de Gantt dinámico, donde cada sprint contiene tareas asignadas por el líder de equipo. Este enfoque facilita la planificación, asignación de responsabilidades y seguimiento del progreso, optimizando la colaboración y eficiencia del equipo.

Tecnologías utilizadas: PHP, MySQL, HTML5, CSS3 y JavaScript.
